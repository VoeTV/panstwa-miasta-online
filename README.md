# 🌍 Państwa i Miasta Online

Multiplayer online game — play "Countries and Cities" with friends in real-time.

## Features

- 🎮 Real-time multiplayer with room codes
- ⚡ Socket.IO powered instant communication
- 🎵 AI Sound Effects (ElevenLabs)
- 📊 Auto-scoring (unique=10pts, duplicate=5pts)
- 🎨 Modern dark UI with Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

Frontend: http://localhost:3000 | Backend: http://localhost:3001
